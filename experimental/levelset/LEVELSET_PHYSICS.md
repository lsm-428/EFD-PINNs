# LevelSet PINN 物理原理与实现约束

> **⚠️ 必须遵守**: 本文件定义了 LevelSet PINN 的正确物理实现，任何修改必须遵循这些约束！

## 一、ψ 的物理定义（核心）

```
ψ(x,y,V,t) = 在 (x,y) 位置，油墨-极性液体界面的 z 坐标高度
```

### 物理分层

```
z=ψ (界面)
│
├─ 油墨区域: 0 < z < ψ
│
└─ 极性液体区域: z > ψ
```

| ψ 值 | 物理含义 |
|------|----------|
| ψ > 0 | 有油墨，值为界面高度 |
| ψ = 0 | 界面在 z=0，油墨被完全推开 = 最大开口 |
| ψ = 常数(>0) | 油墨均匀平铺（如 3μm） |

---

## 二、开口率计算（核心）

### 定义（z=0 平面）

```
在 z=0 平面采样:
  ψ(x,y,0,V,t) > 0 → 有油墨覆盖
  ψ(x,y,0,V,t) ≈ 0 → 界面在此位置 = 开口边缘

开口率 = ψ ≈ 0 的点数 / 总点数 (容差 ε)
```

### 物理过程

| 电压 | 现象 | ψ 分布 |
|------|------|---------|
| **0V** | 油墨平铺底部 | ψ ≈ 常数(3μm) > 0，**无 ψ≈0 点** |
| **逐渐加压** | 油墨开始被推开 | ψ 开始出现 ≈0 的点 |
| **阈值电压** | 刚好能推开油墨 | 首次出现 ψ≈0 的电压 |
| **高电压** | 油墨收缩成岛 | ψ≈0 的点增多 |
| **最高电压** | 油墨几乎全推开 | ψ≈0 的点最多 |

---

## 三、阈值电压

```
阈值电压 V_th: 刚好能把油墨从平铺状态推开的最低电压

物理过程: ψ 从全部 >0  →  开始出现 ≈0 的点
```

---

## 四、体积守恒

### 定义

```
油墨体积 = 油墨区域的体积
         = 对所有 ψ>0 的点，积分 ψ * dx * dy

体积分数 = 油墨体积 / (Lx * Ly * Lz)
```

### 约束

- **总体积守恒**: 油墨总量不变
- **体积守恒误差**: |V_ink(0V) - V_ink(30V)| / V_ink(0V) < 10%

## 六、损失函数约束

### ✅ 必须保留

| 损失项 | 说明 |
|--------|------|
| Data Loss | 拟合训练数据 |
| Continuity | ∇·u = 0 |
| Volume Conservation | 油墨体积守恒 |
| Interface | 界面点 ψ ≈ z |

### ❌ 禁止使用

- `(psi < 0)` 任何形式的判断
- Heaviside 转换用于相分数判断

### ✅ 正确做法

```python
# 开口率
aperture = (np.abs(psi) < epsilon).mean()

# 体积
ink_mask = psi > 0
avg_height = psi[ink_mask].mean()
volume_fraction = avg_height / Lz
```

## 十、常见错误（禁止）

❌ **错误理解**: ψ 是符号距离函数，可正可负  
✅ **正确理解**: ψ 始终 ≥ 0，表示界面高度

❌ **错误**: 用 Heaviside 判断相分数  
✅ **正确**: 直接用 `psi > 0` 和 `psi ≈ 0`

❌ **错误**: 开口率 = ψ > 0 的点占比  
✅ **正确**: 开口率 = ψ ≈ 0 的点占比
