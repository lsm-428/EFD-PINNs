## 一、微流体物理机制

### 1.1 像素微观结构

EWD 像素是一个微米尺度的光流控器件，其垂直结构经过精密设计以实现光开关功能。

```
┌──────────────────────────────────────────────┐
│         顶部透明电极 (ITO Top Electrode)     │
├──────────────────────────────────────────────┤
│                                            │
│        极性液体 (Polar Fluid / Water)      │
│        - 导电 (1MΩ·cm)                      │
│        - 透明                              │
│                                            │
│    ╔═══════════════════════════════╗        │
│    ║   油墨 (Ink / Colored Oil)   ║        │
│    ║   - 绝缘 (Decane)            ║        │
│    ║   - 不透明 (Colored)         ║        │
│    ╚═══════════════════════════════╝        │
├──────────────────────────────────────────────┤
│        疏水层 (Hydrophobic Layer, 400nm)    │ ← Teflon AF
│        - 提供初始疏水性 (θ₀ ≈ 120°)          │
├──────────────────────────────────────────────┤
│        介电层 (Dielectric Layer, 400nm)     │ ← SU-8
│        - 隔离电流，积聚电荷                  │
├──────────────────────────────────────────────┤
│         底部电极 (Bottom Electrode)         │
└──────────────────────────────────────────────┘

  <-- 四周是 20um 高, 5um 宽的墙 (Weir) -->

几何参数 (参考 device_parameters.py):
  Lx = Ly = 174 μm    (像素宽度)
  Lz = 20 μm          (像素高度)
  h_ink = 3 μm        (初始油墨厚度)
  h_polar = 17 μm     (极性液体厚度)
  h_wall = 20 μm      (像素墙高度)
  w_wall = 5 μm       (像素墙宽度)
  d_dielectric = 400 nm (介电层厚度)
  d_hydrophobic = 400 nm (疏水层厚度)
```

**关键材料功能**：
*   **油墨 (Decane)**: 非极性烷烃，染色后作为光阻挡层。自然状态下倾向于铺展在疏水层上。
*   **疏水层 (Teflon AF)**: 具有极低的表面能 ($\gamma_{solid} \approx 18 mN/m$)，使得油墨更愿意附着其上而不是水。
*   **介电层 (SU-8)**: 形成平行板电容器，储存静电能。电压施加在底部电极和极性液体之间。

### 1.2 力学平衡与破裂机制

像素的开关过程是一场**表面张力**与**麦克斯韦应力 (Maxwell Stress)** 的较量。

1.  **关态 (OFF State, 0V)**:
    *   疏水层表面能主导。
    *   系统处于能量最低态：油墨铺满底部，接触角 $\theta_0 \approx 120^\circ$（对水而言是疏水，对油是亲油）。
    *   **现象**：像素显示黑色/彩色。

2.  **破裂过程 (Oil Breaking)**:
    *   施加电压后，极性液体中积聚电荷，与底部电极形成电场。
    *   **麦克斯韦应力**产生向下的静电力，试图拉动水接触绝缘层。
    *   当静电力克服油墨的表面张力时，油膜破裂，对称性被打破。

3.  **开态 (ON State, 30V)**:
    *   静电力主导，固体表面表现为“表观亲水”。
    *   水将油墨挤压到像素角落，接触角 $\theta(V)$ 显著减小。
    *   **现象**：中心区域透明，像素“点亮”。

### 1.3 Young-Lippmann 方程

宏观上，电压对接触角的影响遵循 Young-Lippmann 方程：

$$ \cos \theta(V) = \cos \theta_0 + \frac{\varepsilon_0 \varepsilon_r}{2 d \gamma_{ow}} V^2 $$

*   $\theta(V)$: 施加电压 $V$ 时的表观接触角。
*   $\theta_0$: 初始接触角 (120°)。
*   $\varepsilon_r$: 绝缘层（介电层+疏水层）的有效介电常数。
*   $d$: 绝缘层总厚度 (800nm)。
*   $\gamma_{ow}$: 油-水界面张力 (0.045 N/m)。

> **注意**: 该方程预测 $\theta$ 随 $V^2$ 变化，但在高压下会出现**接触角饱和 (Saturation)** 现象，这是本项目模拟的重点之一。

## 二、Level Set 方法详解

### 2.1 为什么选择 Level Set？

在多相流模拟中（相比于 VOF 方法），Level Set 方法具有独特优势：

1.  **锐利界面 (Sharp Interface)**: 通过零水平集 $\psi=0$ 精确定义界面，而不是模糊的体积分数过渡带。
2.  **几何量计算精确**: 可以直接从 $\psi$ 场计算单位法向量 $\mathbf{n}$ 和曲率 $\kappa$，这对于准确计算表面张力（CSF模型）至关重要。
3.  **拓扑变化自动处理**: 油膜的分裂、合并（如破裂过程）隐含在 $\psi$ 的演化中，无需额外算法。

### 2.2 核心数学定义

我们定义一个高维标量场 $\psi(\mathbf{x}, t)$，它是点 $\mathbf{x}$ 到界面的**符号距离函数 (Signed Distance Function, SDF)**。

$$ \psi(\mathbf{x}, t) = \begin{cases} -d(\mathbf{x}, \Gamma) & \text{in Phase 1 (Ink)} \\ 0 & \text{on Interface } \Gamma \\ +d(\mathbf{x}, \Gamma) & \text{in Phase 2 (Polar Liquid)} \end{cases} $$

### 2.3 几何量计算

利用 $\psi$ 的光滑性，我们可以解析计算关键几何量：

1.  **界面法向量**: 指向 $\psi$ 增加的方向（油 $\to$ 水）。
    $$ \mathbf{n} = \frac{\nabla \psi}{|\nabla \psi|} $$
2.  **平均曲率**: 表面张力的来源。
    $$ \kappa = \nabla \cdot \mathbf{n} = \nabla \cdot \left( \frac{\nabla \psi}{|\nabla \psi|} \right) $$
3.  **平滑 Heaviside 函数**: 用于在相之间过渡物理属性（密度、粘度）。
    $$ H_\epsilon(\psi) = \frac{1}{2} \left( 1 + \frac{\psi}{\epsilon} + \frac{1}{\pi} \sin(\frac{\pi \psi}{\epsilon}) \right), \quad |\psi| \le \epsilon $$
    $$ \rho(\psi) = \rho_{ink} + (\rho_{polar} - \rho_{ink}) H_\epsilon(\psi) $$

## 三、物理模型与数学映射

### 3.1 符号约定 (Sign Convention)

根据 LEVELSET_PHYSICS.md，ψ 是界面高度，始终 >= 0：

| 数学符号 | 物理实体 | 光学性质 | 动力学特性 |
| :--- | :--- | :--- | :--- |
| **$\psi > 0$** | **油墨 (Ink)** | 吸收光线 (黑/红) | 值为界面高度 |
| **$\psi = 0$** | **界面/开口** | 光散射 | 界面在 z=0，最大开口 |
| **$\psi < 0$** | **不允许** | - | 违反物理约束 |

### 3.2 性能指标：开口率

开口率 η 定义为像素在垂直方向上的有效透光面积比例。在 Level Set 框架下，可以通过观测平面（如 z=0）上的积分计算：

$$ \eta = \frac{1}{A_{total}} \int_{\Omega_{2D}} \mathbb{I}(|\psi| < \epsilon) \, dx dy $$

*   **物理意义**: 观测平面上 ψ ≈ 0 的区域面积占比（界面在此位置 = 开口）

## 四、关键约束与损失函数

PINN 的核心在于将物理规律编码为损失函数。以下是本项目中最重要的几个约束：

### 4.1 体积守恒 (Volume Conservation)

**物理直觉**: 油墨是不可压缩流体，在封闭像素内，无论形状如何变化，其总体积必须守恒。
**问题**: 标准 Level Set 方法容易因数值耗散导致"质量损失"（界面向内收缩）。
**实现**:
$$ L_{vol} = \left( \int_{\Omega} \mathbb{I}(\psi > 0) \cdot \psi \, dV - V_{ink}^{initial} \right)^2 $$
强制 ψ > 0 的总体积等于初始油墨体积 ($3\mu m \times 174\mu m \times 174\mu m$)。

### 4.2 符号约束 (Sign Constraint)

**物理直觉**: 油墨必须始终 >= 0（界面高度不能为负）
**实现**: 采用硬约束策略：
*   整个域内: 惩罚 ψ < 0 的情况
*   底部区域 (z < 1.5μm): ψ > 0（油墨存在）
*   顶部区域 (z > 6μm): ψ > 0（油墨存在）
这防止了模型违反物理约束。

### 4.3 Eikonal 约束 (SDF property)

**注意**: 根据 LEVELSET_PHYSICS.md，ψ 不再是符号距离函数，而是界面高度。因此 Eikonal 约束不再适用。

## 五、数值挑战与解决方案

### 5.1 符号错误 (Sign Failure)
*   **现象**: 训练初期，模型可能倾向于输出全正或全负的值，导致界面消失。
*   **对策**: 在训练早期 (Stage 1) 给予 `sign_constraint` 极高的权重，并使用预生成的几何数据进行“热启动”训练。

### 5.2 接触角滞后
*   **现象**: 实验中，油墨收缩和铺展的路径不重合。
*   **对策**: 在边界条件中引入动态接触角模型，区分前进角 $\theta_{adv}$ 和后退角 $\theta_{rec}$。

### 5.3 界面弥散
*   **现象**: 界面附近的物理属性过渡过宽，导致显示对比度下降。
*   **对策**: 使用较小的 $\epsilon$ 参数（界面厚度），并配合 Eikonal 约束保持界面锐度。

## 六、技术实现与网络架构

本项目采用基于 PyTorch 的多尺度 PINN 架构 (`LevelSet3DPINN`)，专门针对 3D 复杂几何界面进行优化。

### 6.1 多尺度网络架构

为了同时捕捉全局流场特征和微小的界面变化，模型采用了三个并行的子网络（Sub-networks）：

1.  **MainNetwork (主网络)**:
    *   **职责**: 学习全局的平滑物理量，如压力场 $p$ 和远离界面的速度场 $\mathbf{u}$。
    *   **结构**: 全连接层 (MLP)，使用 Tanh 激活函数。
    *   **规模**: 典型配置 [64, 64, 32]。

2.  **InterfaceNetwork (界面网络)**:
    *   **职责**: 专注于捕捉 Level Set 场 $\psi$ 的快速变化和界面的尖锐特征。
    *   **结构**: 类似于主网络，但专注于高频分量。
    *   **规模**: 典型配置 [64, 64, 32]。

3.  **CornerNetwork (角点网络)**:
    *   **职责**: 处理像素墙角（Wall/Weir）附近的复杂边界条件和奇异性。
    *   **结构**: 更深或更宽的网络，用于解析几何细节。
    *   **规模**: 典型配置 [64, 64, 64]。

**融合层 (Fusion Layer)**:
三个子网络的输出在最后通过一个融合层进行拼接和线性变换，生成最终的物理场预测。

### 6.2 输入输出张量定义

**输入 (Input) - 6D Tensor**:
模型接受时空坐标和控制参数作为输入：
$$ \mathbf{X} = [x, y, z, V_{from}, V_{to}, t_{since}] $$
*   $x, y, z$: 归一化的空间坐标。
*   $V_{from}, V_{to}$: 电压跳变参数（支持动态电压驱动）。
*   $t_{since}$: 从电压跳变开始经过的时间。

**输出 (Output) - 10D Tensor**:
模型直接输出所有相关的物理场：
$$ \mathbf{Y} = [u_1, v_1, w_1, u_2, v_2, w_2, \psi, p_1, p_2, V_{field}] $$
*   **相1速度 ($u_1, v_1, w_1$)**: 油墨相的速度场。
*   **相2速度 ($u_2, v_2, w_2$)**: 极性液体的速度场。
*   **Level Set ($\psi$)**: 定义相界面的标量场。
*   **压力 ($p_1, p_2$)**: 两相各自的压力场。
*   **电势 ($V_{field}$)**: 像素内部的电势分布，用于计算麦克斯韦应力。

> **双速度场设计**: 相比于单一混合速度场，输出两套速度场能更准确地模拟滑移边界和界面处的速度不连续性。

### 6.3 关键数值方法

1.  **接触角边界条件的有限差分实现**:
    *   **问题**: 在计算边界处的法向导数 $\partial \psi / \partial n$ 时，PyTorch 的自动微分 (Autograd) 有时会断开计算图或产生高阶导数的不稳定性。
    *   **解决**: 在代码中 (`contact_angle_loss`) 采用了**有限差分 (Finite Difference)** 替代方案。通过在 $z=0$ 附近微扰空间坐标，显式计算 $\nabla \psi$，确保了 Young-Lippmann 边界条件的梯度能够正确回传。

2.  **麦克斯韦应力张量 (Maxwell Stress Tensor)**:
    *   不仅计算简单的静电力，而是求解完整的泊松方程 $\nabla \cdot (\varepsilon \nabla V) = 0$ 得到电势场 $V_{field}$。
    *   进而计算麦克斯韦应力张量 $\sigma_M = \varepsilon_0 \varepsilon_r (\mathbf{E} \otimes \mathbf{E} - \frac{1}{2} |\mathbf{E}|^2 \mathbf{I})$ 的散度，得到精确的体积力密度分布。

3.  **光滑 Dirac Delta 与 Heaviside**:
    *   为了在基于网格的方法中处理奇异性，采用了具有紧支集的平滑函数：
        $$ \delta_\epsilon(\psi) = \frac{1}{2\epsilon} (1 + \cos(\frac{\pi \psi}{\epsilon})) $$
    *   这保证了表面张力 $\mathbf{F} = \gamma \kappa \delta \mathbf{n}$ 能够平滑地作用在界面附近的有限宽度区域内，避免了数值震荡。
