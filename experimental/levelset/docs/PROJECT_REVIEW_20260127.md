## 1. 项目概述

### 1.1 核心目标
- **技术目标**: 使用 Level Set 方法替代 VOF 方法，实现更精确的 3D 两相流仿真。
- **关键优势**: 提供精确的界面几何量（法向量、曲率），体积守恒优于传统方法。

### 1.2 技术架构
- **输入**: 6D向量 `[x, y, z, V_from, V_to, t_since]`
- **输出**: 9D向量 `[u1, v1, w1, u2, v2, w2, ψ, p1, p2]`
- **网络**: 多尺度网络 (Main + Interface + Corner)

## 3. 物理建模审查

### 3.1 物理方程 ✅
- **Level Set 传输**: `∂ψ/∂t + u·∇ψ = 0` 正确实现。
- **Navier-Stokes**: 双相流体动量方程，包含表面张力 (CSF) 和电润湿力 (Maxwell应力)。
- **约束条件**: 接触角边界条件、体积守恒约束均已实现。

### 3.2 训练策略 ✅
采用 **分阶段训练 (Multi-Stage Training)**：
1. **Stage 1**: 数据拟合 (Data Loss)
2. **Stage 2**: 连续性约束 (Continuity Loss)
3. **Stage 3**: 完整物理约束 (NS + Transport)
*评价*: 策略合理，有效解决了高维非线性优化的收敛难题。
