# 端到端 PINN（实验性）

这是一个实验性的端到端 PINN 方案，直接从 (x, y, z, t, V) 预测 φ 场。

## 状态

**暂时搁置** - 纯物理约束无法收敛，需要监督数据。

## 文件说明

- `train_end_to_end.py` - 训练脚本
- `visualize_e2e_results.py` - 可视化脚本
- `outputs_e2e_*` - 训练输出目录

## 结论

1. 监督模式（使用 Stage 1 目标数据）可以工作，但与原方案无本质区别
2. 无监督模式（纯物理约束）无法收敛到正确解
3. 原因：电润湿是边界驱动问题，PINN 难以捕捉接触线效应

## 未来方向

- 相场方法（Cahn-Hilliard）
- 接触线动力学建模
- 能量最小化方法
