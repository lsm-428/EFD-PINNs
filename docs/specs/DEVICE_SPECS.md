```python
Lx = 174e-6  # 174 μm - 像素宽度（内沿尺寸）
```

### 叠层结构

```
z = 0 μm
├─ 底层 ITO 玻璃 - 刚性界面
│
├─ 介电层 SU-8 (400nm, ε=3.0) - 电场隔离
│
├─ 疏水层 Teflon (400nm, ε=1.9) - 控制润湿性
│
├─ 围堰 SU-8 (3.5μm 实际 / 20μm 模型)
│  └─ 内部填充 (174×174μm):
│      ├─ 油墨层 (3μm) - 底部，亲疏水层
│      └─ 极性液体层 (17μm) - 顶部
│
└─ 顶层 ITO - 透明电极
```

## ⏱️ 动力学参数

### 二阶欠阻尼系统

```python
tau = 5e-3    # 时间常数 (5ms)
zeta = 0.8    # 阻尼比 (欠阻尼)

omega_0 = 1 / tau  # 自然频率 = 200 rad/s
omega_d = omega_0 * np.sqrt(1 - zeta**2)  # 阻尼频率 = 120 rad/s
```

## 🔬 物理机制

### 电润湿机制

1. **电润湿作用在极性液体上**（不是油墨）
2. **极性液体铺展**，将油墨从像素中心挤向边缘/角落
3. **油墨亲疏水层**（底部 Teflon），不亲围堰壁（相对亲水）
4. **油墨贴底收缩**，形成液滴，不会主动爬墙
5. **翻墙条件**：20V 以上油墨被挤压到极限可能翻墙

### 油墨分布模式

| 开口率 | 模式 | 描述 |
|--------|------|------|
| η < 50% | 中心开口 | 油墨环绕中心透明区域 |
| η > 50% (早期) | 四角液滴 | 毛细管切断，油墨分布在四角 |
| η > 50% (稳态) | 单角液滴 | 油墨汇聚到一个角落 |
