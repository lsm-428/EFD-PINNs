## 🔬 制备工艺

### 完整工艺流程

1. **基底准备**: ITO玻璃清洗
2. **介电层**: SU-8旋涂 (0.4μm)
3. **疏水层**: Teflon AF旋涂 (0.4μm)
4. **表面改性**: 氧等离子体处理 (30s)
5. **围堰**: SU-8厚膜光刻 (20μm) - 在疏水层表面生长
6. **填充**: 油墨注入 (3μm)
7. **封装**: 水溶液填充 + 顶层ITO贴合
