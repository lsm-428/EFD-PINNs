# EFD3D 项目概述

## 项目简介

EFD3D（电润湿像素物理信息神经网络）是一个面向电润湿显示像素的物理信息神经网络（PINN）研究型项目。该项目融合数据拟合与物理约束，支持高效架构、长时训练、性能监控与结果可视化。

## 核心特性

- **物理信息神经网络**：将物理约束显式融入神经网络训练过程
- **多物理场耦合**：支持电润湿像素设计中的几何结构、介电层、流体与电场等多物理耦合
- **高效架构**：支持残差连接、注意力机制等现代神经网络架构
- **动态训练策略**：渐进式训练、动态物理权重调整、混合精度训练
- **完整可视化**：训练曲线、物理约束诊断、性能报告等

## 应用场景

项目面向四类专业场景：

1. **直流阶跃**：电润湿像素的直流电压响应分析
2. **交流频扫**：频率响应特性分析
3. **接触线滞后**：接触线动力学行为研究
4. **温升工况**：温度对电润湿性能的影响分析

## 技术优势

- **物理一致性**：通过物理约束确保模型预测的物理合理性
- **泛化能力**：结合数据拟合与物理约束，提升模型外推稳定性
- **可解释性**：提供详细的物理约束诊断和可视化分析
- **工程实用性**：直接面向工程应用，支持设计决策与参数选型
